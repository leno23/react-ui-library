# 组件概览

Nova UI 当前包含以下组件：

1. 布局：Container、Row、Col、Grid、Space、Divider
2. 基础：Button、Icon、Typography（Title/Text/Paragraph）
3. 表单：Input、Select、Checkbox、Radio、Switch、DatePicker、Form、Upload、Slider、Rate、Calendar、Transfer、Cascader、TreeSelect
4. 反馈：Modal、Drawer、Toast/Message、Tooltip、Popover、Loading、Notification、Tour、Watermark
5. 数据：Table、Pagination、Card、Tag、Badge、Avatar、Progress、Statistic、Descriptions、Empty、Result、Timeline
6. 导航：Tabs、Menu、Breadcrumb、Dropdown、Steps、Collapse、Tree、Anchor、Affix、BackTop

所有组件都支持 `className` 与 `style` 覆盖，并提供类型定义与受控交互。组件体系已逐步对齐 AntD / ElementUI 的中后台常用能力。
